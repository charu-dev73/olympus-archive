# Olympus Archive

A small practice project built while learning HTML and basic web design.

The website is inspired by Ancient Greek mythology and focuses on creating a clean dark aesthetic using simple layouts, images, and typography.

## Pages

- Home
- Gods
- Myths

## Features

- Pure HTML + simple CSS
- Greek mythology inspired visuals
- Classical instrumental background music
- Responsive static layout
- Image-based storytelling sections

## Built With

- HTML
- Inline CSS

## Purpose

This project was mainly created to practice:

- Structuring multi-page websites
- Navigation between pages
- Working with images and media
- Typography and layout design
- Organizing project files
